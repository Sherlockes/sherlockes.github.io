Editando tracks con GPX Editor

2010-09-07
gps
Los archivos GPX son simples archivos de texto con formato XML de forma que para modificar una ruta lo único que hace falta es un editor de texto y algo de tiempo. Es un método simple pero tedioso que funciona y con el que podemos eliminar puntos de la ruta o incluso combinar varias. El problema de este método es que puede acabar con la paciencia de cualquiera, ya que hay que editar el archivo con un programa y luego abrirlo con otro para ver los resultados de la edición con el inevitable sistema de prueba y error.

Time-Lapse / Stop-Motion con VirtualDub

2010-04-28
diseño
Resumiendo bastante, podemos definir Time-Lapse y Stop-Motion como dos técnicas fotográficas que permiten crear un vídeo a partir de una secuencia de imágenes. En el mercado hay muchos programas capaces de hacer esta labor e incluso hay algumos específicos que es lo único que saben hacer. Pero una vez más vamos a utilizar VirtualDub como ya hemos hecho en otras ocasiones y para muy diferentes propositos. Si tienes problemas de visión te aconsejo que maximices el vídeo y lo pauses en cada paso hasta haber entendido todo el contenido del mismo.
